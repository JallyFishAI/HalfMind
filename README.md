# HalfMind

HalfMind is an open-source agent framework that provides a flexible and extensible tooling infrastructure.  
You can precisely define which tools are used, how they are used, and for what purpose, giving you fine-grained control over agent behavior.  
Thanks to this high level of configurability, HalfMind can be used safely and predictably across a wide range of workflows — from simple automations to complex multi-agent systems.
